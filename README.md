# 📝 Taskmate - To-Do List App

Taskmate is a simple and efficient To-Do List web application built using Flask and Python. It allows users to manage daily tasks with ease.

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✏️ Update/edit tasks
- 📋 View all tasks
- 💾 Persistent storage (SQLite)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Jinja Templates  
- **Backend:** Flask (Python)  
- **Database:** SQLite

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/taskmate-flask-todo.git
cd taskmate-flask-todo
```

### 2️⃣ Create virtual environment
```bash
python -m venv venv
source venv/bin/activate   # For Linux/Mac
venv\Scripts\activate      # For Windows
```

### 3️⃣ Install dependencies
```bash
pip install -r requirements.txt
```

### 4️⃣ Run the app
```bash
python app.py
```

### 5️⃣ Open in browser
```
http://127.0.0.1:5000/
```
